= TypeDB Enterprise

== Preliminaries

Connecting to TypeDB Enterprise requires:

* A running TypeDB Enterprise cluster.
* The `address` and `port` of each server node.
* A valid `username` and `password`.
* A valid TLS certificate for in-flight encryption.
