=== Run

// tag::core-run[]
To start a local TypeDB Core server:

[source,console]
----
$ typedb server
----
// end::core-run[]
// tag::core-stop[]
To stop a local TypeDB server, close the terminal where it runs or press kbd:[Ctrl+C].
// end::core-stop[]
